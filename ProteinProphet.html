ProteinProphet

ProteinProphet 通过数据库搜索程序（如SEQUEST）自动验证根据分配给MS/MS光谱的肽进行的蛋白质识别。

    主要功能

    ProteinProphet最重要的是，解决了两个关键问题：
    与"单击"蛋白对应的肽（其他肽对应的肽）不太可能与"多击"蛋白对应。
    蛋白质先知从每个数据集中了解应调整多少肽概率以反映这种蛋白质分组信息。

    许多肽存在于不止一个数据库蛋白质条目中。蛋白质先知在所有相应的蛋白质中分配这些肽，以得出最简单的蛋白质列表来解释观察到的肽。
    ProteinProphet 分析的结果以简洁灵活的方式呈现为 HTML，允许筛选、排序和轻松访问数据库搜索结果。 通常，ProteinProphet 分析遵循 PeptideProphet，计算分配给MS/MS光谱的肽的概率是正确的，在量子实验的情况下，XPRESS 和/或 SAPRatio。

    ProteinProphet 是 ISB 开发的开源 LC/MS/MS 分析管道的组成部分之一

    其他组件包括：
    mzXML 转换器和解析器，摘要 HTML 创建者，交互、XPRESS 定量和 ASAPRatio定量：http://sf.net/projects/sashimi/
    PeptideProphet： http://sf.net/projects/peptideprophet/

      软件使用

      ProteinProphet.pl

      读取多肽和分配的概率，如肽预言所计算的概率，并计算蛋白质存在于原始样本中的概率。

      输出格式可以是：
      XML. 此格式为排序、筛选和查看数据提供了极大的多功能性.UtlilizeXML 变压器程序，如 xsltproc。
      HTML.此格式比 XML 更有限，但不需要 XML 转换器程序（如 xsltproc）。

      作为分析的第一步，从用于搜索与每种肽对应的所有蛋白质的数据库中恢复。这利用了肽预言输出程序。

      运行

      用于使用用户指定的选项运行蛋白质预言的包装器。

        覆盖

        批量覆盖

        读取用于搜索的数据库，以计算所有已识别蛋白质的百分比覆盖率（数据集中观察到的肽表示的蛋白质序列的百分比）。

          帮助图像

          图像

          单击蛋白质预言结果输出上的"帮助"按钮时显示的各种各样的 png 文件。

          CGI计划

          protxml2html.pl

          CGI 程序采用用户的规范来编写样式表（在 XSL 中）， 然后传递给 XML 转换器程序以生成所需的数据视图， 无论是 HTML、选项卡分隔文本还是 XML。这是查看蛋白质预言结果的主力， 在单击"筛选/排序/丢弃选中的条目"，"恢复原始"和"将显示的数据子集写入 Filt"后启动， 并在分析后由 ProteinProphet.pl 启动。

          prot_wt_xml. pl

          CGI程序，显示包含每种肽的所有蛋白质条目。通过单击肽重量启动。

          peptide_html. pl

          CGI程序，显示指定前体离子电荷和肽序列的所有搜索结果。 通过单击前体离子电荷/肽序列启动。

          show_sens_err. pl

          CGI程序，显示蛋白质预言预测的灵敏度和误差率， 作为最小概率阈值的函数。通过单击"敏感/错误信息"启动。

          more_anal.pl

          CGI 程序，显示蛋白质预言分析的详细信息， 以及输入文件和运行设置。 通过点击"更多分析信息"启动。

          show_help.pl

          CGI 程序，显示如何查看蛋白质预言结果的帮助， 显示帮助图像。通过单击"帮助"启动。

          show_dataset_derivation.pl

          CGI 程序，显示用于生成数据子集文件的筛选器。 通过单击"数据集派生信息"启动。

          comet-fastadb.cgi

          CGI程序，显示覆盖在蛋白质序列上的已识别肽。通过点击蛋白质名称启动。

          prot_wt_html.pl

          CGI程序，当蛋白质先知在"HTML"模式下运行时， 显示包含每种肽的所有蛋白质条目。通过单击肽重量启动。

          reviseHTML.pl

          在"HTML"模式下运行时过滤蛋白质预言结果的CGI程序。

          restoreOriginalHTML.pl

          在"HTML"模式下运行时，可恢复过滤的蛋白质预言结果的CGI程序。

          parser

          用于导入每个已识别蛋白质的 XPRESS 和 ASAPRatio 定量信息（当蛋白质预言在 X 或 A 模式下分别运行时） 以及记录用户更新的定量信息（通过单击定量比率）的多功能程序。 程序读取并写入蛋白质预言XML。

          xpress-prophet.cgi

          CGI 程序，显示有助于计算 XPRESS 蛋白质输入定量比的信息。通过点击XPRESS蛋白质比启动。

          xpress-prophet-update.cgi

          使用用户指定的数据更新蛋白质预言XML的CGI程序。通过单击"接受更改"启动。

          asap-prophet-display.cgi

          CGI 程序，显示有助于计算蛋白质进入的 ASAPRatio 定量比的信息。 通过点击 ASAPRAtio 蛋白质比启动。
