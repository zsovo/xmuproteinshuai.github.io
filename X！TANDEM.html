X！TANDEM

X！TANDEM开源软件是一种可以将串联质谱与肽序列进行匹配的软件，这一过程被称为蛋白质鉴定。该软件具有非常简单，复杂的应用程序编程接口（API）：仅在命令行上获取指令的XML文件，并将结果输出到XML文件中，该XML文件已在输入XML文件中指定。此处描述了输出格式（PDF）。此格式用于所有X！系列搜索引擎，以及GPM和GPMDB。 与某些早期的搜索引擎不同，所有X！系列搜索引擎为所有单独的频谱到序列分配计算统计置信度（期望值）。他们还将数据集中的所有肽段分配重新组装到已知蛋白质序列上，并赋予这种组装和比对是非随机的统计置信度。可以在此处找到其公式 。因此，不需要使用单独的组装和统计分析软件，例如PeptideProphet和ProteinProphet。

安装说明：

介绍：

X！Tandem MS Protein Sequence Modeler是一个软件包，其中包括许多c ++和xml文件：
  • 生成可执行文件新版本所需的所有C ++源代码
  • 一个C ++可执行文件（tandem.exe）
  • xml“模板”（default_input.xml）
  • xml分类输入文件（taxonomy.xml）
  • xml测试输入文件（input.xml）
  • 频谱输入测试文件（spectrum_1.pkl）

下载：

下载并解压缩文件
  • tandem-win32-{release-date}.zip
  • tandem-linux-{release-date}.tar.z or
  • tandem-osx-{release-date}.tar.z
如果您使用的是Linux或OS X使用下面的命令来解压缩文件：
tar -xzf tandem-[linux|osx]-tar.z. 这将创建一个名为tandem的文件夹，其中包含3个子文件夹：
  • src - contains the source files for tandem
  • bin - contains the binary for tandem
  • example files and mass spectra
您可以从以下目录的命令行运行tandem，方法是输入：
tandem input.xml或./tandem.exe input.xml（Linux或OS X）

程序的输出将出现在输入中指定的输出文件中。在文件名后附加日期戳的xml文件。如果执行此检查有困难，请检查taxonomy.xml和input.xml中的路径名。阅读default_input.xml文件，以获取有关输入参数文件格式如何工作的更多信息。有关这些xml文件的更多信息，请阅读FAQ。

fasta-包含示例FASTA蛋白序列列表文件。这种类型的文件通常称为“数据库”文件，或简称为“数据库”。但是，它不是任何类型的数据库：它只是具有简单（但指定不当）格式的文件。X！TANDEM不搜索蛋白质序列的数据库。
