PeptideProphet

PeptideProphet自动由SSEQUEST等数据库搜索程序对MS/MS光谱的肽分配。

    主要功能

    从每个集中的数据，它学习正确和不正确的肽中搜索分数和肽属性的分布，并使用这些分布来计算每个结果的正确概率。 相关的肽特性包括与酶裂解相容的乙基数（用于无约束的搜索）、错过的酶裂解数量、前体离子的质量差异、轻或重半胱氨酸的存在（用于 ICAT 实验）以及存在 N-糖基化图案（用于 N-糖基化捕获实验）。 肽先知可用作第二步，在分析从任何类型的质谱仪生成的MS/MS光谱，并使用任意数量的数据库搜索程序分配肽。
    该软件是在西雅图的系统生物学研究所开发的。通常，PeptideProphet分析后是ProteinProphet，它根据相应的蛋白质对肽进行组团，以计算这些蛋白质存在于原始样本中的概率。

    PeptideProphet 是 ISB 开发的开源 LC/MS/MS 分析管道的组成部分之一

    其他组件包括：
    mzXML 转换器和解析器，摘要 HTML 创建者，交互、XPRESS 定量和 ASAPRatio定量：http://sf.net/projects/sashimi/
    ProteinProphet：http://sf.net/projects/proteinprophet/

      软件使用

      评分器

      在 SEQUEST 摘要 html 文件中读取并提取相关的数据库搜索分数和肽属性。 接下来，使用搜索分数，计算每个搜索结果的鉴别分数（fval）。将输出写入 .esi 文件。
      此程序已适应 Mascot 结果（转换为摘要 html 格式），并且必须同样适用于分析其他数据库搜索程序。

      PeptideProphet

      读取包含搜索结果鉴别分数 （fval） 和相关肽属性的 .esi 文件，并迭代地了解这些值在正确和不正确的搜索结果中的分布。 计算数据集中每个搜索结果都正确的概率，并将这些结果写入 .prob 文件

      PeptideProphet输出

      读取摘要 html 文件和 .prob 文件，将写入计算概率作为指定的输出 html 文件的额外第一列

        CGI程序

        plotmodel.cgi

        通过单击特定搜索结果的概率链接触发，输出到肽预言分析。 显示搜索结果鉴别分数 （fval） 和整个数据集中正确和不正确的搜索结果之间的相关肽属性的分布。 叠加在歧视分数分布上是特定搜索结果条目的值。
