Pyteomics的安装

支持Python版本

Pyteomics支持Python 2.7和Python 3.3+。

项目依赖
  • numpy
  • matplotlib
  • lxml
  • pandas
  • sqlalchemy
  • pynumpress
GNU / Linux

获取Pyteomics的首选方法是通过pip Python软件包管理器。新安装的Ubuntu系统的shell代码： 

sudo apt-get install python-setuptools python-dev build-essential
sudo easy_install pip
sudo pip install lxml numpy matplotlib pyteomics
基于Arch的发行版

在Arch Linux和相关发行版上，可以从AUR安装Pyteomics：

  • python-pyteomics
  • python2-pyteomics
Windows

Get pip,如果您还没有的话。

安装Pyteomics及其依赖项：

pip install lxml numpy matplotlib pyteomics
